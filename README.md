## Overview
This repository contains a series of notebooks that progressively build and evaluate LLM-powered chatbot systems and agents. The focus is on experimentation with safeguards, observability, and evaluation frameworks.

### GenAI Basics
* [0-basic-agent](https://github.com/giobbu/notes-ai-agents/blob/main/notebooks/0_basic_agent.ipynb),
* [1-structured-output](https://github.com/giobbu/notes-ai-agents/blob/main/notebooks/1_agent_confidence.ipynb), 
* [2-ollama-websearch](https://github.com/giobbu/notes-ai-agents/blob/main/notebooks/2_ollama_websearch.ipynb), 
* [3-small-chat-llms](https://github.com/giobbu/notes-ai-agents/blob/main/notebooks/3_small_chat_on_sustainability.ipynb), 
* [4-gpt-oss-safeguard](https://github.com/giobbu/notes-ai-agents/blob/main/notebooks/4_ollama_safeguard.ipynb), 

Stack: ollama, websearch-tool, gpt-oss, gpt-oss-safeguard
